# 소개

`@simple-motion-dnd/grid`는 Motion Reorder를 참고해서 만든 드래그 앤 드롭 그리드 라이브러리예요. 하지만 Motion Reorder와는 다른 접근 방식을 사용해요. Framer Motion을 확장해서 만든 컴포넌트기때문에 모바일 환경 터치도 지원하고있어요.

## 드래그 감지 방식

Motion Reorder는 모든 DnD 요소들의 위치 정보를 메모리에 저장해서 드래그가 겹치는 순간을 감지해요. 반면 Simple Motion DnD는 위치 정보를 저장하지 않는 경량화된 구조로 드래그 시점에만 필요한 정보를 동적으로 계산해요.

`elementFromPoint` API를 활용해서 드래그 중 실시간으로 마우스 위치의 요소를 감지해요. DOM API를 직접 활용한 정확한 충돌 검사로 별도의 위치 계산 없이 브라우저 네이티브 기능을 활용할 수 있어요.

## 성능 최적화

위치 정보를 저장하지 않아 메모리 사용량을 최소화하고 드래그 시점에만 필요한 계산을 수행해요. 드래그 이벤트마다 즉시 위치를 계산해서 60fps 애니메이션과 완벽한 동기화를 제공해요.

## 드래그 중 스크롤 지원

드래그 중 컨테이너 경계에 접근하면 자동으로 스크롤되는 기능을 제공해요. 긴 리스트에서도 매끄러운 드래그 앤 드롭 경험을 위해 스크롤 속도와 방향을 실시간으로 조절해요. 스크롤된 상태에서도 정확한 드래그 위치를 계산하기 위해 `window.scrollY`를 고려한 좌표 보정을 통해 다양한 스크롤 상황에서 일관된 동작을 보장해요.

## 기술적 우수성

마우스 커서의 정확한 위치에서 요소를 감지하고 스크롤 오프셋을 고려한 정확한 좌표 계산으로 다양한 화면 크기와 해상도에서 일관된 동작을 제공해요. 아키텍처 설계에서는 최소한의 상태만 유지하고 브라우저의 네이티브 기능을 최대한 활용해서 크로스 브라우저 호환성을 보장해요.

이러한 혁신적인 접근 방식으로 Motion Reorder의 한계를 극복하고, 더 가볍고 효율적인 드래그 앤 드롭 경험을 제공해요.
