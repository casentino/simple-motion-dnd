# 시작하기

`@simple-motion-dnd/grid`는 React와 Motion을 기반으로 한 드래그 앤 드롭 그리드 라이브러리예요. 이 라이브러리를 사용해서 아이템들을 드래그해서 재정렬할 수 있는 그리드 레이아웃을 쉽게 만들 수 있어요.

## 설치

npm 또는 yarn을 사용해서 설치할 수 있어요:

```bash
# npm 사용
npm install @simple-motion-dnd/grid

# yarn 사용
yarn add @simple-motion-dnd/grid

# pnpm 사용
pnpm add @simple-motion-dnd/grid
```
